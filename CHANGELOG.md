# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-11-28

- Initial open source release
- Infinite canvas with pan and zoom via Konva
- Phase-based facilitation palette (Events → Hotspots → Pivotal → Lanes → Actors/Systems → Opportunities → Glossary)
- Sticky note creation with EventStorming visual grammar
- Real-time collaboration via PartyKit and Yjs
- Local persistence with IndexedDB
- Centralized state management with Zustand
- GitHub Pages deployment with custom domain

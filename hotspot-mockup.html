Palette & Cursor Icon Options

The palette shows tool icons at ~24×24px. The cursor uses the same icon when placing shapes. Current hotspot uses a plain red square like other stickies. Here are options for the speech bubble:

Current Palette (for context)
Option A: Mini Speech Bubble (flat)

Simple, recognizable. No rotation at icon size.

Option B: Mini Speech Bubble (tilted)

Matches the tilted shape. More playful.

Option C: Question Mark Bubble

Conveys "question/problem" semantics directly.

Option D: Exclamation Bubble

Conveys "alert/problem" - hotspots are pain points.

Option E: Filled Bubble

Uses red fill like other stickies. More visible.
